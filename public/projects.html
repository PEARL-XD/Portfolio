Spotify Clone: A Music Streaming Web Application


This project is a Spotify Clone, built using HTML, CSS, JavaScript, Node.js, and Express.js. The app replicates the core features of a music streaming service, allowing users to browse songs, play tracks, and enjoy an immersive music experience. I implemented dynamic song loading through Node.js, which enables a smooth user interface with minimal delay.

  • User Authentication: Built a login/logout system using local storage to store and verify user credentials, ensuring that only authenticated users can access the songs.
  • Dynamic Song List: The song list is dynamically rendered on the browser through Node.js to ensure scalability and efficiency, especially for a limited set of songs.
  • Frontend: Designed a clean and responsive user interface using HTML and CSS, ensuring an optimal experience across devices.
  • Backend: Utilized Express.js to set up routes and manage server-side logic, creating a seamless integration between the frontend and the backend.

Currency Converter: A Real-Time Currency Conversion Web Application


This project is a Currency Converter, built using HTML, CSS, and JavaScript. The app allows users to convert currencies in real-time and view exchange rates for all countries. The data is fetched dynamically from an external Exchange Rate API, ensuring accurate and up-to-date conversion rates.

  • Currency Conversion: The app allows users to select from a list of currencies and input an amount to convert it to another currency, displaying the converted value instantly.
  • Real-Time Exchange Rates: Integrated the Exchange Rate API to fetch the latest exchange rates, ensuring the app displays up-to-date values for all supported currencies.
  • Country Flags Integration: Flags of countries are displayed next to the currency names, making the app visually intuitive for users and enhancing the user experience.
  • JavaScript Integration: Utilized JavaScript to dynamically update the displayed exchange rates and handle currency conversion logic, allowing for a smooth and interactive user experience.

    • This project showcases my ability to work with external APIs to retrieve and display real-time data, while creating a clean and responsive front-end user experience.

Typing Master Game: A Typing Speed and Accuracy Web Application


This project is a Typing Master Game, built using HTML, CSS, and JavaScript. The game challenges users to improve their typing speed and accuracy by typing random paragraphs within a set time. Real-time performance metrics, such as words per minute (WPM), characters per minute (CPM), and mistakes, are dynamically displayed to enhance the learning experience.

  • Dynamic Text Selection: The game randomly selects a paragraph from a predefined set, ensuring a varied typing experience for each session.
  • Real-Time Typing Metrics: The game tracks and updates WPM, CPM, and mistakes as users type, providing immediate feedback on their performance.
  • Timer Integration: A countdown timer runs from 60 seconds, challenging users to complete typing the text within the allocated time.
  • Error Tracking and Feedback: Mistyped characters are highlighted in red, and correct characters are marked in green, helping users quickly identify and learn from their mistakes.
  • Responsive Design: Designed a mobile-friendly and responsive interface using HTML and CSS, ensuring users have a smooth and engaging experience across all devices.
  • JavaScript Logic: Utilized JavaScript to handle the dynamic interaction between the timer, input field, and typing feedback, ensuring smooth and real-time updates.

This project highlights my ability to create interactive web applications with real-time data updates and user feedback, showcasing my skills in JavaScript for interactive functionalities and front-end development for building responsive, user-friendly interfaces.

Task Manager: A Simple Task Management Web Application


This project is a Task Manager web application, developed using HTML, CSS, and JavaScript, which allows users to add, delete, and edit tasks. The app also includes a simple login/logout system, ensuring that only authenticated users can manage their tasks. The user credentials are stored locally with a constant password for simplicity.

  • Task Management: Users can add new tasks, delete completed tasks, and edit existing tasks to keep track of their to-do list efficiently.
  • Login/Logout System: Implemented a basic login page where users must enter a constant password to gain access to the task manager, with a logout feature to secure the app.
  • Dynamic Task List: The task list is dynamically updated as users add, edit, or delete tasks. This provides a smooth and interactive experience.
  • Responsive Design: Designed the interface using HTML and CSS, ensuring a simple, clean, and mobile-friendly layout for a smooth user experience across devices.
  • JavaScript Logic: Utilized JavaScript to handle the addition, deletion, and editing of tasks. The login/logout functionality is also controlled via JavaScript to ensure proper session management.

This project highlights my ability to build a functional web application with user authentication and dynamic content management. It demonstrates my skills in front-end development with HTML, CSS, and JavaScript, as well as my understanding of building interactive and user-oriented web apps.
